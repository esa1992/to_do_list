Cross Todo - Windows Portable

Quick start:
1) Unzip the archive to any folder (for example: C:\Apps\cross_todo).
2) Open the folder and run cross_todo.exe.
3) Sign in or create an account.

If your network requires a proxy:
1) On the login screen, click the network icon in the top-right corner.
2) Open "Network / Proxy".
3) Enter proxy host, port, login, password and save.
4) Click "Check connection".
5) If TLS errors appear in a corporate network, enable "TLS compatibility mode".

Notes:
- Installation is not required.
- To remove the app, just delete the unzipped folder.
- On first launch, Windows SmartScreen may show a warning:
  click "More info" -> "Run anyway".
